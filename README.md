# Web Chat Room

---------------------------------------------------------------------------

**A Web Chat Room Project writing in Java, you can chat with others whom in the LAN.**

# How To Use

1. 运行ChatServer.java
2. 运行ChatClient.java
3. 客户端出现登录界面
4. 登录成功后进入通信面板
